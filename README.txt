INSTRUKCJA

1.Instalacja.
Uruchom program arkanopongClient-1.0.exe i postępuj zgodnie z dalszymi instrukcjami.
Uruchom program arkanopongServer-1.0.exe i postępuj zgodnie z dalszymi instrukcjami.

2.Uruchamianie.
Uruchom serwer, a następnie czterech klientów.
Podaj w polu tekstowym adres IP klienta i kliknij connect.

3.Rozgrywka.
Aby wypuścić piłkę, wciśnij spację.
Poruszaj paletkę klawiszami strzałek.
Punkty zdobywa się za zbijanie bloków, a traci się za przepuszczenie piłki ze swojej krawędzi ekranu.
